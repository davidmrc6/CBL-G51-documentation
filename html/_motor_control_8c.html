Motor initialization.

Initialize motor properties, setting motor pins to output. Indirectly sets the PWM frequency to 100. Frequency = PWM base frequency / (pwmSetClock value * pwmSetRange value) PWM base frequency = Raspberry Pi frequency - 19.2MHz

Parameters
[in]motorMotor struct pointer
[in]indexUseful for debugging
[in]MFForward pin
[in]MBBackward pin
[in]en1PWM pin
Returns
void

The function sets the PWM mode to mark-space (PWM_MODE_MS) which is commonly used for motor control as it provides a consistent frequency and duty cycle. This mode is suitable for applications that require precise control over the PWM signal, such as driving motors.
