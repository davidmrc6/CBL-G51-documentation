CBL-G51

This is the GitHub repository for the Embedded Systems CBL project of Group 51.

Overview

This repository contains software developed for a line following robot which is able to sort discs into different buckets. The robot is meant to simulate an embedded system which would be implemented in international airports, able to take in some luggage (in our case, smalled 3D printed discs) and follow a predifined path towards a terminal. The source directory contains libraries for the different sensors that make the core mechanisms of the robot possible, such as the TCS34725 color sensor for detecing the color of the 3D printed discs, the TCRT5000 infrared sensors which are used to implement the line following algorithm, the SG90 servo motor which pushes the discs onto the terminals, and the motors used to push the robot forward (and backward). The line following algorithm is robust and optimized for efficiency, such that the robot is able to follow the line consistently while minimizing the chances of it derailing. The code is well-documented and easily replicable given the presence of the required hardware, as it has been tested thoroughly in diverse scenarios to ensure reliability.

Requirements

Hardware

  • RaspberryPi 3
  • TCS34725 color sensor
  • TCRT5000 infrared sensor module
  • SG90 servo motor (SG90Servo)
  • L298n motor driver
  • Motors (MotorControl)
  • AA batteries
  • Power source for the RPi3 (Powerbank)
  • Fischertechnik piecces
  • Connecting wires

Software

  • the wiringPi.h GPIO library for C
  • standard C libraries: stdio.h, signal.h, stdlib.h
  • the pthread.h library for multi-threading
  • cmake version 3.25 or above

Instructions

  • On local machine, navigate to a directory and open it in terminal
  • Run git clone https://github.com/davidmrc6/CBL-G51.git to clone the directory onto your local machine
  • Open the terminal and navigate to the project path with cd /.../src
  • Run the following commands to make and compile the project files:\ mkdir build\ cd build\ cmake ..\ make
  • You should have now created an executable file, which you can run with ./src
  • To safely interrupt the program, click CTRL + C in terminal
